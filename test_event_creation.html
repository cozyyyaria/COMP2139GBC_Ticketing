Test Event Creation

This page will help you test event creation:

Steps to test:

  1. Go to: http://localhost:5128/Event/Create
  2. Fill out the form with these sample values:
    • Title: Summer Concert 2025
    • Event Date: 2025-06-15T19:00
    • Category: Select "Online Webinar" or "Online Class"
    • Ticket Price: 25.99
    • Available Tickets: 100
  3. Click "Save"
  4. You should be redirected to the Events list
  5. Check that your event appears in the list

If it doesn't work:

Check the browser console (F12) for any JavaScript errors and look for validation error messages on the form.

Test the search/filter features:

  1. Go to: http://localhost:5128/Event
  2. Try typing in the search box
  3. Try filtering by category
  4. Try changing the sort order
